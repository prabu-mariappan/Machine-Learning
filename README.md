Naming Conventions
------------------

First 4 Chars - xxML (Fixed Chars)
Fifth Char    - _
Sixth Char    - Algorithem name
Seventh Char  - _
Eight Char onwards - small Desc

EXAMPLE
=======

xxML_LR_Ecommerce Customers

LR stands for Linear Regression
